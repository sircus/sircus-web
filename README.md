# Sircus Web

Gulping Handlebars  and JSON.

## Building

Install : `nodejs` `npm` `gulp`

```
$ git clone https://github.com/sircus/sircus-web.git
$ cd sircus-web
```

build:

```
npm install
gulp build
```

Watch & Connect:

```
gulp
```
